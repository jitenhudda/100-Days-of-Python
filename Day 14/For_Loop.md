## Day 14 - For Loop

A `for` loop is used to iterate over a sequence or a range of values.

## Key Concepts

- Used when the number of iterations is known.
- Works with `range()`, strings, lists, tuples, sets, and dictionaries.
- Executes a block of code for each item in a sequence.
- Commonly used for tables, traversing data, and repetitive tasks.

## Key Points

- `for` loop iterates over a sequence.
- `range()` is commonly used with `for` loops.
- Strings can be traversed character by character.
- Works with list, tuple, set, and dictionary.
- Use `for` loop when the iteration count is known.
- Use `while` loop when the iteration count is unknown.

## Quick Revision

A `for` loop is used to iterate over sequences and is preferred when the number of iterations is known.
