## Day 06 - Identifiers

Identifiers are names given to variables, functions, classes, and other objects in Python.

## Key Concepts

- Identifiers help uniquely identify program elements.
- An identifier can start with an alphabet (`A-Z`, `a-z`) or underscore (`_`).
- Identifiers cannot start with a digit.
- Special characters are not allowed except underscore (`_`).
- Keywords cannot be used as identifiers.

## Key Points

- Python is case-sensitive.
- `name` and `Name` are different identifiers.
- `_name` is a valid identifier.
- `first_name` is a valid identifier.
- `2name` is an invalid identifier.
- Keywords like `True`, `False`, `class`, and `import` cannot be used as identifiers.

## Quick Revision

Identifiers are names used for variables, functions, classes, and objects. They must follow Python naming rules and cannot be keywords.
