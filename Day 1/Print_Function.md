## Day 01 - Print Function

Used to display output on the screen.

## Key Concepts

- `print()` displays text, numbers, and variables.
- Multiple values can be printed using commas.
- Python adds a space between comma-separated values by default.
- `sep` is used to change the separator between values.
- `end` is used to change the default line ending.

## Key Points

- Default `sep = " "`
- Default `end = "\n"`
- `sep="/"` prints values separated by `/`.
- `sep="-"` prints values separated by `-`.
- `end=" "` prints the next output on the same line.

## Quick Revision

`print()` displays output, `sep` changes the separator, and `end` controls the line ending.
