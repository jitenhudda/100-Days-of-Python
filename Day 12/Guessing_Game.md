## Day 12 - Guessing Game

A simple game where the user guesses a randomly generated number.

## Key Concepts

- `random.randint()` generates a random number within a given range.
- User input is used to make guesses.
- A `while` loop runs until the correct number is guessed.
- Conditional statements provide hints to the user.
- A counter tracks the number of attempts.

## Key Points

- Random number range: `1` to `100`.
- `while` loop continues until the guess is correct.
- If guess is smaller, user is asked to guess higher.
- If guess is larger, user is asked to guess lower.
- Attempts are counted using a counter variable.

## Quick Revision

The Guessing Game combines random numbers, user input, loops, conditions, and counters to create an interactive program.
