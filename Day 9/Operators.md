## Day 09 - Operators

Operators are used to perform operations on values and variables.

## Key Concepts

- Arithmetic Operators perform mathematical calculations.
- Comparison Operators compare two values and return `True` or `False`.
- Logical Operators work with boolean values.
- Bitwise Operators work on binary numbers.
- Assignment Operators update variable values.
- Identity Operators check memory locations.
- Membership Operators check whether a value exists in a sequence.

## Key Points

- Arithmetic: `+`, `-`, `*`, `/`, `%`, `**`, `//`
- Comparison: `<`, `>`, `<=`, `>=`, `==`, `!=`
- Logical: `and`, `or`, `not`
- Bitwise: `&`, `|`, `^`, `~`, `<<`, `>>`
- Assignment: `=`, `+=`, `-=`, `*=`, `/=`
- Identity: `is`, `is not`
- Membership: `in`, `not in`
- Python does not support `++` and `--` operators.

## Quick Revision

Operators are special symbols used to perform arithmetic, comparison, logical, bitwise, assignment, identity, and membership operations.
