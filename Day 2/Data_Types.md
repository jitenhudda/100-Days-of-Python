## Day 02 - Data Types

Data types define the type of value stored in a variable.

## Key Concepts

### Basic Data Types

- `int` → Stores whole numbers.
- `float` → Stores decimal numbers.
- `bool` → Stores `True` or `False`.
- `complex` → Stores complex numbers.
- `str` → Stores text or characters.

### Container Data Types

- `list` → Ordered and mutable collection.
- `tuple` → Ordered and immutable collection.
- `set` → Unordered collection of unique values.
- `dict` → Stores data in key-value pairs.

## Key Points

- Integer values can be very large.
- Float values contain decimal points.
- Boolean has only two values: `True` and `False`.
- Strings can be written using single, double, or triple quotes.
- Lists use `[]`.
- Tuples use `()`.
- Sets use `{}` and remove duplicates.
- Dictionaries use `{key: value}` format.

## Quick Revision

Python data types are mainly divided into Basic Data Types (`int`, `float`, `bool`, `complex`, `str`) and Container Data Types (`list`, `tuple`, `set`, `dict`).
