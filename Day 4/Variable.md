## Day 04 - Variables

Variables are used to store data in memory.

## Key Concepts

- Python does not require variable declaration before use.
- Variables can store different types of data.
- Python follows Dynamic Typing.
- A variable can be reassigned with a different data type.
- Multiple variables can be assigned in a single line.

## Key Points

- Dynamic Typing → No need to specify data types.
- Dynamic Binding → Same variable can store different data types.
- Variable values can be updated anytime.
- Multiple assignment is supported.
- One value can be assigned to multiple variables.

## Quick Revision

Variables store data in Python. Python uses Dynamic Typing and Dynamic Binding, allowing variables to change values and data types during execution.
