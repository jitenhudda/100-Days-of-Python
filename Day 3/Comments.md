## Day 03 - Comments

Comments are used to explain code and improve readability.

## Key Concepts

- Comments are ignored by Python during execution.
- They are used to make code easier to understand.
- Single-line comments start with `#`.

## Key Points

- Comments do not produce any output.
- Comments help in code documentation.
- Single-line comments use `#`.
- Multi-line comments are commonly written using triple quotes (`'''` or `"""`), but they are technically strings, not true comments.

## Quick Revision

Comments are used to explain code. Python ignores comments during execution.
