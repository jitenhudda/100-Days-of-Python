## Day 10 - If Else

`if`, `elif`, and `else` are used to make decisions in a program.

## Key Concepts

- `if` checks a condition.
- `elif` checks another condition if the previous one is false.
- `else` runs when all conditions are false.
- Conditions return either `True` or `False`.
- Logical operators can be used to combine conditions.

## Key Points

- `if` is checked first.
- `elif` is optional and can be used multiple times.
- `else` is optional and executes when no condition matches.
- `and` requires all conditions to be true.
- Decision-making is a core concept in programming.

## Quick Revision

`if`, `elif`, and `else` are used to execute different blocks of code based on conditions.
