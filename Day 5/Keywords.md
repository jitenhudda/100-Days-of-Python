## Day 05 - Keywords

Keywords are reserved words that have special meaning in Python.

## Key Concepts

- Keywords cannot be used as variable names.
- Python is a case-sensitive programming language.
- Keywords are predefined by Python.
- Examples: `True`, `False`, `break`, `class`, `import`.

## Key Points

- Keywords have special purposes in Python syntax.
- Their meaning cannot be changed.
- Python provides the `keyword` module to view all keywords.
- `keyword.kwlist` returns the list of Python keywords.

## Quick Revision

Keywords are reserved words in Python that have predefined meanings and cannot be used as identifiers.
