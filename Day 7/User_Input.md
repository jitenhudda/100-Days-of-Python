## Day 07 - User Input

User input allows users to enter data while the program is running.

## Key Concepts

- `input()` is used to take input from the user.
- Input makes programs interactive and dynamic.
- By default, `input()` returns data as a string.
- `type()` is used to check the data type of a variable.

## Key Points

- Static software only displays information.
- Dynamic software takes input and provides output.
- `input()` always returns a string unless converted.
- Adding two inputs without conversion results in string concatenation.
- `type(variable)` shows the variable's data type.

## Quick Revision

`input()` is used to take user input, and the returned value is stored as a string by default.
