## Day 13 - Range Function

The `range()` function generates a sequence of numbers.

## Key Concepts

- Commonly used with loops.
- Generates numbers within a specified range.
- Supports start, stop, and step values.
- The stop value is not included in the output.

## Key Points

- `range(stop)` → Starts from `0`.
- `range(start, stop)` → Starts from a specified value.
- `range(start, stop, step)` → Uses a custom increment/decrement.
- Negative step values generate numbers in reverse order.
- `list(range())` converts the sequence into a list.

## Quick Revision

`range()` generates a sequence of numbers using start, stop, and optional step values.
