## Day 15 - Nested Loop

A nested loop is a loop inside another loop.

## Key Concepts

- The inner loop executes completely for each iteration of the outer loop.
- Commonly used for pattern printing.
- Useful for working with rows and columns.
- Can be created using `for` loops, `while` loops, or both.

## Key Points

- Outer loop controls rows.
- Inner loop controls columns.
- Widely used in star, number, and shape patterns.
- Execution starts from the outer loop and then the inner loop.
- More nesting increases the number of iterations.

## Quick Revision

A nested loop is a loop inside another loop and is mainly used for pattern printing and row-column based problems.
